To execute a PHP script when this event is fired, just save a PHP file in this folder and it will be included when the event occurs. Files are executed in alphabetical order.

NOTE:
See also invoice_created (this event is fired when invoices are generated by the system, but invoice_created is fired when an invoice is created by an administrator).

EVENT:
invoices_generated

PARAMS:
["number_generated"] => int (Total number of invoices that were generated)
["ids"] => int[,int[,int...etc.]] (Comma-separated list of the ID numbers of the newly generated invoices)
