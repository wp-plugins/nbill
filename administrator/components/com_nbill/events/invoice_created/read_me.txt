To execute a PHP script when this event is fired, just save a PHP file in this folder and it will be included when the event occurs. Files are executed in alphabetical order.

NOTE:
See also invoices_generated (this event is only fired when an invoice is created manually by an administrator - invoices_generated is fired when invoices are generated by the system).

EVENT:
invoice_created

PARAMS:
["id"] => int (The newly created invoice id number - you can look up the rest of the details on the #__nbill_document and #__nbill_document_items tables)
